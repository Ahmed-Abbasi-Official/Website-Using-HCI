HCI UI Design Using Norman, GOMS & KLM

Appointment Booking, Form Submission & Medicine Dose Input

1. Medicine Dose Input UI

User Goal: Enter correct medicine dosage

Feedback: Dose submitted successfully.

Norman’s 7 Stages – Dose Input

  1. Goal: Enter correct dose
  2. Intention: Use dose input form
  3. Specify: click input -> type -> submit
  4. Execute: User type name and value -> press submit
  5. Perceive: Confirmation message
  6. Interpret: Dose saved correctly
  7. Evaluate: Goal achieved

GOMS – Dose Input

Goal: Submit correct dosage

Operators: Click, Type, Submit

Method: Select field → type → submit

Selection Rule: Keyboard preferred for expert users

KLM – Dose Input

StepOperator
